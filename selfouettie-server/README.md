# Development

### Run server

```sh
runMain net.yoshinorin.selfouettie.HttpServer
```

### Stop Server

Press `Enter` key.

# Code Format

On IntelliJ `Ctrl + shift + l`

# Test

Run test.

```
$ sbt test
```

Generate coverage report.

```
$ sbt coverageReport

or

$ sbt clean coverage test coverageReport
```
